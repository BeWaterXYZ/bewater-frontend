# bewater-website

This repo is managed via [PNPM](https://pnpm.io).

## Requirements
Node >= 16.5
pnpm >= 7

## Recipes

* Install dependencies
  * `pnpm install`
* Start dev website
  * `pnpm dev`
* Build website
  * `pnpm build`
* Export website
  * `pnpm export`
* Lint everything
  * `pnpm lint`
* View UI Components in Storybook
  * `pnpm storybook`
